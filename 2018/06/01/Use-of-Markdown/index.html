Markdown

Markdown 是一个 Web 上使用的文本到HTML的转换工具，可以通过简单、易读易写的文本格式生成结构化的HTML文档。目前 github、Stackoverflow 等网站均支持这种格式。——Markdown

对齐方式

在表头下方的分隔线标记中加入 :，即可标记下方单元格内容的对齐方式：

  • :--- 代表左对齐
  • :--: 代表居中对齐
  • ---: 代表右对齐

页面内跳转

  1. 页面内定义一个锚，及要跳转到的位置

    <span id="1">XXX</span>

  2. 使用如下格式定义链接即可

    [跳转到XXX](#1)
